# To-Do-Web-Application
A simple and interactive To-Do List web application built using HTML, CSS, and JavaScript to help users manage daily tasks efficiently.

This is a simple and interactive To-Do List web application made using HTML, CSS, and JavaScript. It helps users manage their daily tasks in an easy way.

##  Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Pending / Completed)
- Task counter
- Data saved using Local Storage

##  Technologies Used
- HTML
- CSS
- JavaScript

##  How It Works
1. Enter a task in the input box
2. Click the "Add" button or press Enter
3. Task will appear in the list
4. Click on task to mark it complete
5. Click "X" to delete the task

##  Storage
All tasks are saved in the browser using local storage, so data is not lost after refresh.

##  Responsive Design
This app works on both mobile and desktop screens.
